# basd4g.hatenablog.com

My [blog](https://ya0201.hatenablog.com)'s articles.

## Installation

```sh
$ git clone https://github.com/basd4g/basd4g.hatenablog.com.git
$ cd basd4g.hatenablog.com
$ gimonfu init
```

## Download articles

```sh
$ gimonfu pull
```

## Upload articles

```sh
$ gimonfu push
```
